# Haftstories landing

Static landing page for Cloudflare Workers Static Assets.

## Cloudflare build settings

- Build command: `npm install`
- Deploy command: `npm run deploy`
- Root directory: `/`
- Production branch: `main`

The custom domain should be added in Cloudflare under:
Workers & Pages → haftstories → Settings → Domains & Routes → Add → Custom Domain

Use `haftstories.com` as the main domain.
