HAFTSTORIES — CLOUDFLARE PAGES

1. Open Cloudflare → Workers & Pages.
2. Choose Create → Pages → Upload assets.
3. Upload the ZIP file or the folder containing index.html.
4. After deployment, open Custom domains.
5. Add haftstories.com.
6. Optionally add www.haftstories.com and redirect it to the main domain.
